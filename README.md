# QYay App!

## Overview
QYay! is a live Q&A application designed to streamline audience engagement during events. It allows event organizers to collect and manage questions in real-time.

## Tech Stack
- **Frontend:** Next.js
- **Backend:** Express.js, jsScript
- **Database:** firebase
